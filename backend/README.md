# SerenMind Backend

Spring Boot REST API for SerenMind mental wellness application.

## Tech Stack
- Java 17+
- Spring Boot 3.x
- MySQL 8.0
- JPA/Hibernate
- Flyway Migrations
- JWT Authentication
- OpenAI Integration
